## О приложении

**TripTrack — Road Journal** — бесплатное iOS-приложение (iPhone, iOS 17+) для автоматической записи автомобильных поездок.

Ключевые факты:
- Автоматическая запись маршрутов в фоне
- Маршрут окрашивается по скорости (зелёный < 50, жёлтый 50-90, оранжевый 90-110, красный > 110 км/ч)
- Лента поездок с группировкой по месяцам
- Фото и заметки привязанные к точке на маршруте
- Fog of War карта — открываешь территорию как в игре
- 30+ бейджей и 30 уровней водителя, стрики
- Гараж с несколькими машинами, учёт топлива
- Live Activity на экране блокировки во время записи
- 100% офлайн, все данные на устройстве, без облака, без аккаунта
- Открытый исходный код: https://github.com/OneZee23/trip-track-ios
- Бесплатно навсегда, без подписок
- Создатель: OneZee, инди-разработчик из Краснодара, 71 000 км на одометре
- App Store: https://apps.apple.com/us/app/triptrack-road-journal/id6760650361

## Контекст и позиционирование

- Google убил Timeline в 2025, люди потеряли историю местоположений — TripTrack это замена
- Polarsteps — для путешественников/отпусков, не для ежедневных поездок
- MileIQ — для бизнес-пробега, не для воспоминаний
- Fog of World — похожее приложение с fog of war, но без фокуса на поездках/авто
- Strava — для спортсменов, не для водителей
- TripTrack — единственное приложение именно для обычных водителей которые ездят каждый день

## Целевая аудитория

- Автовладельцы 25-45 лет
- Ежедневные коммьютеры
- Любители поездок на выходные
- Дорожные путешественники
- Люди кто скучает по Google Timeline

## Тон

- Дружелюбный, без понтов
- Не «мы создали революционное приложение», а «я сделал приложение которое сам хотел»
- Инди-дух, открытый исходный код, прозрачность
- Честность: приложение молодое, но развивается активно

## Задание: создай следующие файлы

### 1. `4pda-post.md` — Пост для форума 4PDA

Раздел: iOS → Программы → Навигация/GPS

Формат:
- Заголовок: TripTrack — дневник автомобильных поездок [Free, iOS 17+]
- Описание приложения (3-5 абзацев, что делает, зачем нужно)
- Список ключевых функций (буллеты)
- Скриншоты: укажи плейсхолдеры [скриншот: экран записи], [скриншот: лента поездок] и т.д.
- Ссылка на App Store
- Ссылка на GitHub (открытый код!)
- Тон: технический но доступный, как обычный пост на 4PDA

### 2. `habr-article.md` — Статья для Хабра

Тема: «Как я сделал дневник поездок после того, как Google убил Timeline»

Формат:
- Длинная статья (1500-2500 слов)
- Структура: проблема → поиск решений → решил сделать сам → технические решения → что получилось → планы
- Хабы: iOS, Swift, Разработка мобильных приложений, Геоинформационные технологии
- Технические детали: CoreLocation, MapKit, CoreData, Kalman filter для GPS, Live Activity, SwiftUI
- Упомянуть open source и пригласить контрибьюторов
- Тон: разработчик рассказывает другим разработчикам, но интересно и не занудно
- Обязательно упомянуть: fog of war (geohash-based), speed-colored routes, badge/XP система
- В конце: ссылки на App Store, GitHub, сайт, Telegram

### 3. `drive2-post.md` — Пост для Drive2

Тема: «Сделал приложение-дневник для тех, кто много ездит»

Формат:
- Пост в бортжурнал или блог
- Разговорный стиль, от первого лица
- Рассказать историю: много езжу Краснодар-побережье, Google убил Timeline, ничего не нашёл, сделал сам
- Показать реальные поездки (упомянуть Геленджик→Краснодар, горный серпантин)
- Объяснить fog of war простыми словами («как в играх — карта закрыта туманом, каждый километр его снимает»)
- Упомянуть что бесплатно и приватно
- Ссылка: https://trip-track.app?lang=ru
- Тон: как пост обычного водителя, не реклама

### 4. `reddit-post.md` — Пост для Reddit

Subreddit: r/iOSapps или r/apple

Тема: «I built a free, open-source drive diary app after Google killed Timeline»

Формат:

- Show HN / side project стиль
- Упомянуть: open source, no account, no cloud, fog of war
- Ссылки на App Store и GitHub
